#### Smart Plant Pot (스마트 화분 관리 시스템)

> 아두이노 기반 임베디드 시스템 창의 프로젝트  
> 자동 급수 + 환경 센서 + 블루투스 제어로 식물 관리의 편리함을 제공합니다.

---

#### 개요
현대인들은 바쁜 일상 속에서 식물을 꾸준히 관리하기 어렵습니다.  
본 프로젝트는 아두이노와 여러 센서를 활용하여 자동으로 물을 주고, 환경을 관리하는 스마트 화분 시스템 을 개발하는 것을 목표로 합니다.  
이 시스템은 토양 습도, 온도, 조도를 측정해 자동으로 물을 공급하거나 LED 조명을 제어합니다.

---

#### 개발 목적
- 식물 관리의 자동화 및 편의성 향상
- 물 낭비 방지 및 생장 효율 개선
- 임베디드 시스템 제어 능력 향상
- 실제 생활에 응용 가능한 프로젝트 설계 경험 습득

---

#### 주요 기능

| 기능 | 설명 | 사용 부품 |
|------|------|------------|
| 환경 측정 | 온도, 습도, 조도 실시간 감지 | DHT11, LDR |
| 자동 급수 | 토양 습도에 따라 펌프 자동 제어 | 토양 습도 센서, 워터 펌프, 릴레이 |
| 조명 제어 | 조도 부족 시 LED 점등 | 고휘도 LED |
| 상태 표시 | LCD에 환경 상태 표시 | 16x2 LCD |
| 수동 제어 | 스마트폰 앱으로 직접 제어 | HC-06 블루투스 모듈 |

---

#### 하드웨어 구성

| 구성 요소 | 역할 |
|------------|------|
| Arduino UNO | 메인 제어 보드 |
| DHT11 | 온습도 센서 |
| LDR | 조도 센서 |
| Soil Sensor | 토양 습도 센서 |
| Relay + Pump | 자동 급수 장치 |
| 16x2 LCD | 상태 표시 |
| HC-06 | 블루투스 통신 |
| LED | 조명 제어용 |

---

#### 시스템 동작 흐름

```mermaid
flowchart TD
    A[센서 데이터 측정] --> B{습도/조도 임계값 확인}
    B -->|습도 낮음| C[펌프 ON (급수)]
    B -->|조도 낮음| D[LED ON (조명)]
    B -->|정상| E[대기 상태 유지]
    A --> F[LCD에 상태 표시]
    F --> G[블루투스로 상태 전송]
```

---

#### 개발 환경
- 개발 보드: Arduino UNO
- 언어: C / Arduino C
- IDE: Arduino IDE
- 통신: Bluetooth (HC-06)
